# 关于泛型的问题:

# 泛型是什么
在大型项目中函数,类,接口灵活处理多种数据类型的一种方式

# 泛型的意义
- 当你的函数、接口或类将处理多种数据类型时；
- 当函数、接口或类在多个地方使用该数据类型时

# 泛型接口
```js
interface Person<T, U> {
    name: T;
    age: U
}

let p1: Person<string, number> = {
    name: '1',
    age: 1
}
```

# 泛型类
```js
class User<T> {
    age: T;
    constructor(age: T) {
        this.age = age;
    }
}

const u = new User('1')
console.log(u.age);
```

# 泛型函数
```js
function test<T> (a: T): T {
    return a; 
}

console.log(test(1));
```
